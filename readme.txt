# Interactive Grid Application

This application allows users to create and interact with a grid using a graphical user interface (GUI).
The grid can be customized in terms of size, and users can add walls, set a start point, set a goal point, and create a maze.
The grid can be interacted with using the mouse.

## Features

- Create a grid of a specified size
- Add walls to the grid
- Set a start point on the grid
- Set a goal point on the grid
- Create a maze on the grid
- Print the grid
- Choose a search technique (BFS, DFS, A*)

## How to Run

1. Ensure you have Python installed on your machine.
2. Install the required libraries: tkinter, customtkinter, and json.
3. Run the `maze.py` file.

## Usage

- Use the "Width" and "Height" fields to specify the size of the grid.
- Click the "Create Grid" button to create the grid.
- Use the "Walls", "Start", "Goal", and "Create Maze" buttons to interact with the grid.
- Use the "Print Grid" button to print the current state of the grid.
- Use the "Search Technique" dropdown to select a search technique.
